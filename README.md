```
  - index.html : HTML 참고 사이트로 이동하는 url을 넣은 html 사이트 코드
      a 태그에서 href를 통해 사이트 지정.
      코드상에는 없지만 한 페이지에서 base태그를 이용하여 특수한 위치 혹은 홈페이지로의 이동하도록 코딩할 수도 있다.
      (id로 한 페이지 내 이동도 가능)
      
  - index2.html : 표 만들기
      style 코드를 통해 background, width 지정 가능
      아마 p태그 내부에서 text 위치 지정은 text-align 코드로 가능
      
      tr : 행 만들기
      td : 열 만들기
      th : 헤더 만들기(볼드체)
  
      <tr>  
        <td></td>
        <td></td>
        <td></td>
      </tr>
      
      ㅡㅡㅡㅡ 1행을 만듬
      l l l    "1행 안에 들어갈" 3열을 만듬
      
   - index3.html : 이미지 표 만들기
      <ing src="이미제주소">를 하면 웹에 있는 이미지 불러오기 가능
      
   - index4.html : pre 태그 및 유니코드 활용
      pre 태그는 내가 쓰고 싶은대로 쓸 수 있음.
      &#숫자 형식의 유니코드를 통해서 특수기호 작성 가능 -> 굳이 유니코드로 안 적어도 작성 가능
      
   - index5.html : 오디오 넣기
   - 아래의 형태로 웹에 있는 음원 넣기 가능
   
      <audio controls autoplay loop>
        <source src="웹음원.mp3" type="audio/mpeg">
      </audio>
    
```
